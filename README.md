# excerpts-extractor
Script to extract excerpts from https://github.com/ERC-TibSchol/TEI-curation
This tool was created in the framework of the project The Dawn of Tibetan Buddhist Scholasticism (11th-13th c.) (TibSchol). Cf. [https://www.oeaw.ac.at/ikga/tibschol](https://www.oeaw.ac.at/projects/tibschol/home) for more information.
